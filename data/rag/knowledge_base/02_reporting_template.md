# Reporting Template for Cyber Range Sessions

## Session Metadata
- Tenant/User ID
- Agent ID
- Session ID
- Objective
- Target scope

## Timeline
- Time
- Action summary
- Evidence reference
- Result

## Findings
- Finding title
- Affected target in scope
- Evidence references
- Lab impact summary

## Failed Paths
- Attempted path
- Why it failed
- What changed in next decision

## Final Notes
- What was validated
- What remains unverified
- Recommended next lab step
